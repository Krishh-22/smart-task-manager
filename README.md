# Smart Task Manager (Python CLI)

Smart Task Manager is a command-line task management application built with Python.  
It allows users to create, manage, and track tasks with priorities and deadlines while storing data permanently using JSON.

This project demonstrates modular programming, file persistence, and task analytics.

---

## Features

- Add new tasks with priority and deadline
- View all tasks
- Mark tasks as completed
- Delete tasks
- Search tasks by keyword
- Detect overdue tasks automatically
- View task completion statistics
- Persistent storage using JSON

---

## Project Structure

task_manager/
│
├── main.py        # Main application logic
├── task.py        # Task class and task-related methods
├── storage.py     # JSON storage handling
└── tasks.json     # Stores all tasks
